# 從基準文本到正文定位

這裏解釋從基準文本開始，到生成基準文本樹，到最後爲基準文本定位的整個過程，及其中的一些細節。

## 生成原始文本（raw text）

- 在網上找可用的文本，如 <a href="https://github.com/JasonWade001/chtxt">JasonWade001/chtxt</a>
- 以 <a href="https://github.com/wingmingchan64/CanonicalTextTrees/blob/main/tools/php/生成原始文本.php">生成原始文本.php</a> 去除格式、去除標點符號、規範用字

## 手編基準文本

- 按一定的格式，編輯原始文本，以編成基準文本
	- 篇目在第一行
	- 段與段、章與章、闕（如宋詞）與闕之間，加入一行空行
	- 段/章/闕中可以適當分行
- 如果時間容許，以可靠版本原文，校對基準文本

## 生成基準文本樹

- 以 <a href="https://github.com/wingmingchan64/CanonicalTextTrees/blob/main/tools/php/生成raw_text.php">生成raw_text.php</a> 去除格式、去除標點符號、規範用字
